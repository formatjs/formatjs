## Usage

### Simple

```tsx
import '@formatjs/intl-segmenter/polyfill.js'
```

### Dynamic import + capability detection

```tsx
async function polyfill(locale: string) {
  if (shouldPolyfill()) {
    await import('@formatjs/intl-segmenter/polyfill-force.js')
  }
}
```
