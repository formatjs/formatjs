## Requirements

- [`Intl.Collator`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/Collator)
- [`Intl.DateTimeFormat`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/DateTimeFormat) or [polyfill](intl-datetimeformat)
- [`Intl.NumberFormat`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/NumberFormat) or [polyfill](intl-numberformat).

## Usage

### Simple

```tsx
import '@formatjs/intl-supportedvaluesof/polyfill.js'
```

### Dynamic import + capability detection

```tsx
async function polyfill() {
  // This platform already supports Intl.supportedValuesOf
  if (shouldPolyfill()) {
    await import('@formatjs/intl-supportedvaluesof/polyfill.js')
  }
  // Alternatively, force the polyfill regardless of support
  await import('@formatjs/intl-supportedvaluesof/polyfill-force.js')
}
```
